Demo using Terraform, EC2, S3 and GitHub Actions

## Demo Overview

This Demo demonstrates how to:

- Create **AWS EC2 instance** with SSH access
- Create **S3 bucket** to host a static website
- Store website code in **GitHub**
- Use **Terraform** to provision AWS resources
- Automatically deploy website from GitHub to S3 using **GitHub Actions**
- Keep S3 bucket in sync with repository changes

---

## Tools Used

- HTML (`index.html`)
- Terraform
- AWS (EC2, S3)
- GitHub & GitHub Actions
